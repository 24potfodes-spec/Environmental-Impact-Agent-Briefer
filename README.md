## 🎯 Problem Statement
Environmental Impact Briefer

Given a product, industry, or activity, the system:
- Analyzes environmental impact  
- Estimates carbon footprint  
- Identifies sustainability issues  
- Suggests eco-friendly alternatives  

---

## 🤖 Why an Agentic Approach

| Requirement | Single Prompt Limitation | Agent-Based Solution |
|------------|------------------------|---------------------|
| Real-world data | Static knowledge | Uses research-based inputs |
| Structured analysis | Unorganized output | Multiple agents handle tasks |
| Sustainability insights | Generic suggestions | Focused environmental reasoning |
| Accuracy | No validation | Multi-step processing |

---

## 🏗 System Architecture
Multi-agent pipeline:

1. Researcher Agent → Collects environmental data  
2. Extractor Agent → Filters key insights  
3. Writer Agent → Generates structured report  

---

## 🔄 Agent Workflow

**Step 1 — User Input**  
User enters product / activity  

**Step 2 — Research**  
Researcher gathers environmental data  

**Step 3 — Extraction**  
Extractor filters useful insights  

**Step 4 — Report Generation**  
Writer creates structured report
